# 项目运行

## 方式1

该项目已经上传云端，[点击访问](http://47.110.133.39:8003/)

## 方式2

* 下载项目中的`密码学课设exe`压缩包
* 解压后双击文件夹下的`ECC.exe`
* 打开浏览器输入`127.0.0.1:8003`即可访问

## 方式3

* 下载源码
* 确保电脑中的python>=3.5(本人使用的是python3.9.0)
* 进入文件主目录，在cmd执行pip install requirements.txt
* 然后执行python app.py
* 打开浏览器输入`127.0.0.1:8003`即可访问
